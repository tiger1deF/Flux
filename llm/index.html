Overview

The LLM module provides a comprehensive framework for working with language models, including inference, embeddings, and model management. The module is organized into three main components:

Base Models

  • Base LLM interface for model inference

  • Parameter and configuration management

  • Thread-safe execution handling

See Language Models for implementation details.

Embeddings

  • Multiple embedding providers (Gemini, Pulse)

  • Async and sync embedding generation

  • Vector operations and caching

See Embeddings for implementation details.

Inference

  • Multiple model providers (Gemini, Pulse, Azure)

  • Async and sync inference

  • Configurable generation parameters

  • Thread-safe model management

See Inference for implementation details.
