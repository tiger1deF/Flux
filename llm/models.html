Language Models

Overview

The language models module provides base classes and utilities for working with LLMs, including parameter management, type checking, and thread-safe execution.

Base LLM

Parameter Models

Models for LLM parameter handling and embedding function wrappers.

This module provides dataclasses and base classes for handling LLM parameters and embedding functions with support for both synchronous and asynchronous operations.
