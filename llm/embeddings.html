Embeddings

Overview

The embeddings module provides implementations for various embedding models, with support for both synchronous and asynchronous operations. Each implementation is thread-safe and handles proper resource management.

Base Embedding Function

class llm.models.BaseEmbeddingFunction(embedding_fn, dimension=None, cleanup_fn=None)[source]

Bases: object

Base class for wrapping embedding functions with standardized interface.

Provides a unified interface for both synchronous and asynchronous embedding functions, with support for batching, type inference, and output formatting.

Parameters:

  • embedding_fn (Callable) – The embedding function to wrap

  • dimension (Optional[int]) – Optional fixed dimension of embeddings
